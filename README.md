# Sunrise Farm

A cozy 3D browser farm game built for mobile-first play: plant wheat, harvest crops,
care for animals, ride Hazel, drive the tractor, and grow the farm into Millbrook.

## Itch build

- Uploadable HTML zip: `sunrise-farm-itch.zip`
- Release media: `media/itch/`
- Launch notes and draft fields: `docs/LAUNCH.md`
